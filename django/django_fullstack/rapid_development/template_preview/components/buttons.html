🔘 Buttons Component Preview

All Bootstrap button styles and variations ready to copy and paste into your Django templates.

🎯 Primary Action Buttons

<button type="submit" class="btn btn-primary">Create</button> <button type="submit" class="btn btn-success">Save</button> <button type="submit" class="btn btn-warning">Update</button> <button type="submit" class="btn btn-danger">Delete</button>

🔘 Secondary Buttons

<button type="button" class="btn btn-secondary">Cancel</button> <button type="button" class="btn btn-outline-primary">Edit</button> <button type="button" class="btn btn-outline-danger">Delete</button>

📏 Small Buttons

<button type="submit" class="btn btn-sm btn-primary">Add</button> <button type="submit" class="btn btn-sm btn-outline-danger">Remove</button>

👥 Button Groups

<div class="btn-group" role="group"> <button type="button" class="btn btn-outline-primary">View</button> <button type="button" class="btn btn-outline-warning">Edit</button> <button type="button" class="btn btn-outline-danger">Delete</button> </div>

🧭 Navigation Buttons

Go Back Add New Logout
<a href="/" class="btn btn-secondary">Go Back</a> <a href="/create" class="btn btn-primary">Add New</a> <a href="/logout" class="btn btn-outline-light">Logout</a>

⚠️ Confirmation Buttons

<button type="submit" class="btn btn-danger" onclick="return confirm('Are you sure you want to delete this item?')"> Delete </button>

🚫 Disabled Buttons

<button type="submit" class="btn btn-primary" disabled>Processing...</button>
📋 How to Use These Buttons:
  1. Copy the code from any button you like
  2. Paste it into your Django template
  3. Customize the text and actions as needed
  4. Add Django template variables like {{ item.name }} if needed

💡 Tip: All buttons are styled with Bootstrap 5.3.0 and work perfectly with Django forms.
